# log-anomaly-detector


## Overview
A system-level tool that analyzes log activity patterns and detects abnormal
behavior using simple statistical techniques.

## Motivation
System logs are critical for monitoring security incidents and system health.
This project demonstrates how anomalies can be detected without complex
machine learning models.

## Features
- Log activity aggregation
- Statistical anomaly detection
- Explainable output

## Technologies Used
- Python
- Operating Systems concepts
- Basic statistics

## Use Cases
- Server monitoring
- Security incident detection
- System health analysis

## Status
Self-learning / academic project
